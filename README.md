# LyricsLabMuse
## Get started
- Clone the repository
- OPENAI_API_KEY  
    Right now, the project is configured to work with OpenAI models. To configure OpenAI API Key:
    1. Create .env file
    2. Add key like this
    ```
    OPENAI_API_KEY=you_api_key
    ```
- Install requiered libraries
    ```
    pip install -r requirements.txt
    ```
